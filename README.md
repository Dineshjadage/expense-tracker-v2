# 💰 SpendSmart — Personal Expense Tracker Pro

**Internship Project 1 (v2.0) — iStop Web Development Front-End Mentorship**

---

## 📌 Project Overview
A fully upgraded Personal Expense Tracker with a professional sidebar layout, multi-page navigation, analytics dashboard, budget planner, and CSV export.

---

## 🛠️ Technologies Used

| Technology       | Purpose                                          |
|------------------|--------------------------------------------------|
| **HTML5**        | Semantic structure, modal, multi-tab layout      |
| **CSS3**         | Sidebar, animations, responsive grid, transitions|
| **Bootstrap 5**  | Modal, grid, utilities, responsive classes       |
| **JavaScript**   | All logic, localStorage, CSV export, charts      |
| **jQuery**       | DOM manipulation, event delegation, AJAX-style   |
| **Chart.js**     | Bar, Doughnut, Line, and Pie charts              |

---

## ✨ Features (v2.0)

### Dashboard
- ✅ 4 Summary cards: Income, Expenses, Balance, Daily Average
- ✅ Monthly Income vs Expense **bar chart**
- ✅ Expense by Category **doughnut chart**
- ✅ Recent transactions list

### Transactions
- ✅ Full transaction history with delete
- ✅ Search by description or category
- ✅ Filter by category, type (income/expense)
- ✅ Sort by newest, oldest, highest, lowest amount

### Analytics
- ✅ Monthly spending **trend line chart**
- ✅ Income vs Expense **pie chart**
- ✅ **Category breakdown** with visual progress bars
- ✅ **Top 5 expenses** list

### Budget Planner
- ✅ Set monthly spending limits per category
- ✅ Visual progress bars (green → amber → red)
- ✅ Over-budget alerts
- ✅ Budgets saved to localStorage

### Other
- ✅ **Month filter** — view any month or all time
- ✅ **Export CSV** — download all transactions
- ✅ Add transaction **modal** with note field
- ✅ Toast notifications for all actions
- ✅ Data persists with **localStorage**
- ✅ Fully responsive (mobile friendly)

## 🚀 How to Run

1. Extract the ZIP file
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
3. No installation, server, or internet required!
